## Joystick for Obsidian

The plan for the Joystick plugin was to give the user more granular controls over keyboard interactions when writing outlines in Obsidian.

However, [the devs have said they're looking into building some of the key pieces into core](https://discord.com/channels/686053708261228577/817515900349448202/822433686191734795) (that is a link to the private #mobile channel, so you may not have access to it yet), so development on Joystick is paused for now.
